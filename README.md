## Create a simple String calculator with a method signature like this:

- Input: a string of comma-separated numbers
- Output: an integer, sum of the numbers

### Examples:

- Input: “”, Output: 0
- Input: “1”, Output: 1
- Input: “1,5”, Output: 6

###  How to run
 
- RUBY version - 3.3.4 
- project_path/script/string_calculator.sh
